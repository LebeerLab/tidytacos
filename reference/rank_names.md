# Return rank names associated with a tidytacos object if these are defined. In case of undefined rank names, the function returns the taxon_id field.

Return rank names associated with a tidytacos object if these are
defined. In case of undefined rank names, the function returns the
taxon_id field.

## Usage

``` r
rank_names(ta)
```

## Arguments

- ta:

  a tidytacos object
