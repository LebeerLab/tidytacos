# Return a visualization designed for a small number of samples

Return a visualization designed for a small number of samples

## Usage

``` r
tacoplot_zoom(ta, sample = sample_id, n = 15, nrow = NULL)
```

## Arguments

- ta:

  A tidytacos object.

- sample:

  A string, representing the unique sample name of interest

- n:

  An integer, representing the amount of colors used to depict taxa

- nrow:

  An integer, representing the amount of rows in the facet_wrap
