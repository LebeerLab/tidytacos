# An S4 class to store a grouped tidytaco with [group_samples](https://lebeerlab.github.io/tidytacos/reference/group_samples.md)

This S4 class stores the output from
[group_samples](https://lebeerlab.github.io/tidytacos/reference/group_samples.md).

## Slots

- `groups`:

  the different grouped tidytacos.
