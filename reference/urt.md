# Upper Respiratory Tract samples

Human microbiome samples from the upper respiratory tract (URT), taken
from a paper by De Boeck et al. It contains nose as well as nasopharynx
samples. Most samples were taken using a swab method, but a minority was
taking with the aspirate method.

## Usage

``` r
urt
```

## Format

### `urt`

A tidytacos object with 1,947 taxa and 207 samples:

- participant:

  individual participant ID

- location:

  nose (N) or nasopharynx (NF)

- method:

  swab (S) or aspirate (A)

- plate:

  which of the three plates sequenced the sample is located on
